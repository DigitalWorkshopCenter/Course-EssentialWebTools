Assignment #2

Overview

Today, you will connect via SSH to a remote server, and do some work on it.

Your tasks

  1. Create an account at SDF.ORG.

  2. Follow the instructions in the email to connect to your remote server.

  3. Search Google if you come across any errors while trying to connect (note, you will not be able to see your password as you type it. And you will most likely need to type it instead of trying to paste it).

  4. Try typing some commands. What does help bring up?

  5. Execute each of the commands that help suggested, and go through them while exploring. Have fun while you learn!

Submission

  1. Take 5 screenshots of different things you tried with the ssh server.

  2. Put these screenshots into a folder on your computer that is connected to a github repository that you made in the previous lesson.

  3. Commit your changes (your images) and push your changes to Github.

  4. Fill out This Form to submit your assignment. You will need to include a link to your github repository in it so that the instructor can see the screenshots you took.
