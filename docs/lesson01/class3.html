Hands-on Exercise 3
Create new Website and publish it with Github Pages

Overview

In this exercise, you will create a new and awesome website, then publish it to the web for anyone to see using Github Pages.

Instructions

Create and Link Repository

  • Go to Github.com and create a new repository called webProject. This repository should be public.

  • Copy the link that shows up after you create the repo

  • Do the same thing that you did in class exercise 1 this evening to connect it to a folder on your computer.

Create your awesome website

  • In the folder, create the following files:

    • index.html

    • style.css

    • script.js

    • aboutMe.html

  • Add some css to the style sheet, some JavaScript to the script file, and some content to both html pages. There should be a button to call a javascript function. There should also be links on each page to navigate to the other page.

  • Once you're happy with your website, use git to commit and push your changes to github.

Deploy website to Github Pages

  • Go to the repository page on github.com and click on Settings.

  • In the menu bar on the left hand side, toward the bottom, click Pages

  • In the None dropdown, click it and select master

  • Click save

  • Click on the link that appeared above. This is where your website will be deployed. It can take a few minutes for it to show up.

Conclusion

Congratulations. You've just deployed your first web application. :)
