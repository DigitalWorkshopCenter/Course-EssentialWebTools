Course Description

The course content found in these webpages was developed by Nathan Birch to fulfill the Performance Based Objectives of the Essential Web Tools course developed and managed by the Digital Workshop Center educational institution.

Instructor Introduction

Performance Based Objectives

Upon successful completion of the Essential Web Tools class, students will be able to:

  • Effectively use modern code editors and related plugins

  • Understand the Command Line

  • Implement and understand Git, and related version control tools

  • Discuss backup tools and the importance of backups

  • Introduction to package managers

  • Understand and use browser developer tools

Course Pre-requisites

Coming into this course

  • Basic digital literacy including file management is required.

  • Web Design with HTML & CSS is required
