Hands-on Exercise 1
Install NodeJS

Overview

In order to use npm as a package manager, we need to install Node.js. So in this exercise that is exactly what we are going to do.

Instructions

  1. Navigate to The Node.js Download page

  2. Select the correct download for your operating system.

  3. Verify that the installation completed and was successful:

    • Open the terminal in VS Code:

    • In the terminal, execute the following command: 

      npm version

      You should see something similar to the following output:

    • Notice, that our computer recognizes npm commands as its way to work with node.js on our computer.

Congratulations. :) You have successfully installed Node.js. Now, we can use npm commands to manage packages and download code onto our computers.
