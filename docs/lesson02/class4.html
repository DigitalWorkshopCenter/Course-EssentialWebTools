Hands-on Exercise 4
VS Code Hotkeys

Overview

For this activity, we are going to do some awesome stuff with VS Code hotkeys

Instructions

  • Download this zip folder of images. Open it and make note of how many images there are.

  • Make a numbered list in Excel or Google Sheets that will have enough numbers to match the images in the zip file you downloaded.

  • Paste this list of numbers in the body of an html file.

  • Using hotkeys, add the prefix and suffix to each number (img1.jpg, img2.jpg, etc.), Then, add an image element for each one of the numbers in this list. Some helpful hotkeys may be:

    • Add Cursor Above/Below

    • Add Selection to Next Find Match

    • Select All Occurrences of Find Match

    • Add Cursors to Line Ends

    Tip: Your ctrl, option and arrow keys will come in handy for handle numbers of different character lengths.

  • Make it a goal to produce valid html to create image elements for each image all at once using hotkeys. Get the html file to successfully run and show all images in the browser.
