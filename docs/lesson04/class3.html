Hands-on Exercise 3
Adding Dependencies to our Browserify Application

Overview

For this exercise, we are going to build on our browserify app that we made in the last activity. When you download this zip file, you'll need to run npm install in the terminal of the folder to install all of the dependencies listed in the package.json. In short, this will create the node_modules folder. Then, run the command: 

browserify main.js -o bundle.js

Instructions

  1. Let's start by adding a css/js framework called Bootstrap to our main.js file. 

    require('bootstrap/dist/js/bootstrap');
require('./node_modules/bootstrap/dist/css/bootstrap.css');

  2. Now, lets beef up Browserify a bit and install the browserify-css package. This will allow us to easily use css packages from our node_modules folder. Execute the following command in the terminal: 

    npm install --save browserify-css

  3. We also need to install bootstrap: 

    npm install --save bootstrap

  4. Finally, let's recreate bundle.js with the changes in our main.js file and with our newly installed packages. Run this command in the terminal: 

    browserify main.js -g browserify-css -o bundle.js

  5. Go ahead and run this in the browser. It's difficult to say whether or not the bootstrap package is working, so let's add some bootstrap css into our html.

  6. Navigate to bootstrap.com and copy the first navbar code by clicking the copy button:
    Ensure that the bootstrap version that shows in the url bar of the browser matches the bootstrap version number in your package.json.

  8. Save the file, and check your index.html file in the browser. You should see something like this: You can see that the bootstrap css is working by the style of the navbar. You can also see that the js is working because of the dropdown button. Now, try resizing the page to represent a mobile view. You should see a nice collapsed navbar provided by bootstrap:

  9. As a fun exercise, open your package.json file and delete the line of code that looks like "browserify": "^17.0.0",.
    Then, delete the node_modules folder from your project directory. Finally, execute the command npm install. This will reinstall all the dependencies listed in your package.json. So it won't include the browserify package this time, just the browserify-css package.

Conclusion

Congratulations. You have successfully added more dependencies to your project and it is looking awesome. Now that you are using browserify-css you'll be able to add pretty much any js or css dependency that you want.
