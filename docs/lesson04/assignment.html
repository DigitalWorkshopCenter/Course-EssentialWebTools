Assignment #4

Overview

Your final assignment will cover all of the Performance Based Objectives of this course. You will build a web application that utilizes several npm packages. You will gain experience using your text editor and related plugins, using the terminal/command line in VS Code, working with git and uploading your code there (which also function as a backup), using package managers and 3rd party packages and using the developer tools throughout this process.

Instructions

Create a New Repository

  1. Go to github.com and create a new repository.

  2. Create a new folder on your computer that is separate from all other git repositories on your machine.

  3. Open VS Code and on the Welcome Page select Clone Git Repository. In the prompt provided, paste the url from the new git repository you made.

Create the Application

  1. Create the following html files

    • index.html

    • account.html

    • about.html

    • contact.html

    • partial_nav.html

  2. Create a JavaScript file called script.js and a css file called style.css

Add some npm packages to make this project awesome

  1. Initiate npm to be able to use node packages: Open the terminal for this project and execute the command: 

    npm init

  2. You may install all of these packages separately, or you can do it all at once like this: 

    npm install --save browserify-css jquery bootstrap particlesjs

  3. In script.js you will add the following code:

  4. jquery

    1. You are going to use jquery to put the navbar on each page. You will write the code for the navbar (or rather, use Bootstrap's navbar that you modify) in a single file called partial_nav.html. Then, you will use jquery in your script.js file to load it onto the active page. 

      var $ = require('jquery');
$(document).ready(function () {
    $('#navMenu').load('./partial_nav.html');
});

    2. Include the following code in the body of each other html file where you want your navbar to appear: 

      <div id="navMenu"></div>

  5. browserify-css

    1. Run the following command to package all js files into the browserify bundle: 

      browserify script.js -g browserify-css -o  bundle.js

  6. bootstrap

    1. Add the following code to script.js: 

      require('bootstrap/dist/js/bootstrap');
require('./node_modules/bootstrap/dist/css/bootstrap.css');

    2. With that there, you can now start adding bootstrap components to your application.

    3. In partial_nav.html retrieve a bootstrap navbar that you like. Copy and paste it into your html file. Modify the links in the navbar to go to your about, account, contact and index pages.

    4. In the home page, put a bootstrap carousel there with several images.

    5. In the about page, use a bootstrap accordion with at least three sections.

    6. In the contact page include a bootstrap form to let people contact you.

    7. In the account page, include a bootstrap table with some information that you might see on a generic account page.

  7. particlesjs

    1. Put the following line of code into your partial_nav.html above the opening nav tag. 

      <canvas class="background"></canvas>

    2. In script.js include the following at the top of the file: 

      let Particles = require('particlesjs');

    3. Add the following code to your style.css file: 

      html,
body {
  margin: 0;
  padding: 0;
  z-index: 1;
  background-color: black;
}

.background {
  position: absolute;
  display: block;
  top: 0;
  left: 0;
  z-index: 0;
}

    4. Add the following code to script.js to start the particles: 

      let particleHome = {
  selector: '.background',
  connectParticles: true,
  speed: 5,
  color: ['#7FFF00', '#DC143C', '#1E90FF'],
};
Particles.init(particleHome);
