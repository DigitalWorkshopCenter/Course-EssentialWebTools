Hands-on Exercise 2
Modifying Code and Updating Remote Repository

Overview

This exercise will start where we left off with the first exercise today.

In this exercise, we will create several files and update our Github repository.

Instructions

Creating Files

  1. Create a file called index.html.

  2. Create a file called intro.html.

  3. Create a link in index.html to go to intro.html

Updating Your Remote Repository

  1. Click on the source control button in VS Code

  2. Enter a commit message, something like "created files for hands on exercise 2"

  3. Press cmd or ctrl + enter or click on the check mark

  4. Click on the three dots, and select push.

  5. Go to github, and verify that your changes have been pushed to github.

  6. Click on commits and look through the different commits you've made in your repository

Conclusion

Congratulations! You have just done everything that you need to do in order to update your repository in Github. You will follow this same process anytime you finish an assignment, want to share code with someone, finish a task in a project, etc.
