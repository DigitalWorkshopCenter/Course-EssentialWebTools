Hands-on Exercise 1
Create your First Repository

Overview

The focus of this assignment is to create our first git repository. Here are our goals for today:

  1. Find code on your computer that you've written previously

  2. Initialize a git repository in the folder that contains that code

  3. Push this code to a new remote repository on Github

Instructions

  1. Find code on your computer that you've written previously (Perhaps from your HTML and CSS class)

  2. Go to Github.com and create a new repository for this code.

  3. Open a new VS Code window and select Open Folder from the file menu. Select the folder on your computer with this code.

  4. You should see your files in the file menu. Now, click on the Source Control icon (below the magnifying glass).

  5. Select Initialize Repository

  6. Type a commit message and click the check mark.

  7. Click the three dots above the commit input box, and click push.

  8. You'll see an error message appear saying that there isn't a remote connected. Click the button to add a remote now.

  9. In the input box provided, paste the github url of your new repository and click Add remote from URL.

  10. Enter a remote name, generally origin but it can be anything you like.

  11. Now that you've added the remote (or connected your local folder to a Github repository), you can now push to it. Click on the three dots again, and click push.

  12. If you see a popup about a master upstream branch, just select okay.

  13. Now, go to Github, view your repository, and verify that all of your code is present.
